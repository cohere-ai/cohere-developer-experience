Rerank provides a powerful semantic boost to the search quality of any keyword or vector search system.

<Steps>
### Setup
This quickstart example uses the `cohere` Python SDK. Install with the following command:

```bash
pip install -U cohere
```

Next, import the library and create a client.

### Reranking

Then, perform reranking by passing the documents and the user query to the Rerank endpoint. We can also pass the `rank_fields` parameter. The model will rerank based on order of the fields passed in.

<Tabs>
<Tab title="Cohere Platform">
```python PYTHON
# Add the user query
query = "Any email about check ins?"

# Rerank the documents
results = co.rerank(
    model='rerank-english-v3.0',
    query=query,
    documents=emails,
    top_n=2,
    rank_fields=["from", "to", "date", "subject", "body"]
)

for result in results.results:
    print(result)
```
