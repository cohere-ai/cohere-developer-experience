To maximize convenience in building on and switching between Cohere-supported deployment platforms, we have developed SDKs that seamlessly support whichever platform you choose.

## SDK Support by Platform

The table below summarizes the platforms in which you can use Cohere's SDKs. You'll notice it contains many links; the links in the "sdk" column take you to Github source for the language-specific SDKs, while all the others take you to example snippets for setting up the SDKs on the respective platforms.
