We're excited to announce the release of **Command A Vision**, Cohere's first model capable of understanding and interpreting visual data alongside text. This groundbreaking addition to our Command family brings enterprise-grade vision capabilities to your applications with the same familiar API interface you know and love.

## Key Features

### Multimodal Capabilities
- **Text + Image Processing**: Seamlessly combine text prompts with image inputs
- **Enterprise-Focused Use Cases**: Optimized for business applications like document analysis, chart interpretation, and OCR
- **Multilingual Support**: Officially supports English, Portuguese, Italian, French, German, and Spanish

### Technical Specifications
- **Model Name**: `command-a-vision`
- **Context Length**: 128K tokens
- **Maximum Output**: 8K tokens
- **Image Support**: Up to 20 images per request (or 20MB total)
- **API Endpoint**: Chat API

The API structure is identical to our existing Command models, making integration seamless:

```python
import cohere

co = cohere.Client("your-api-key")

response = co.chat(
    model="command-a-vision",
    messages=[
        {
            "role": "user",
            "content": [
                {
                    "type": "text",
                    "text": "Analyze this chart and extract the key data points",
                },
                {
                    "type": "image_url",
                    "image_url": {"url": "your-image-url"},
                },
            ],
        }
    ],
)
```
