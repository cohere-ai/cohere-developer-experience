Today, we're announcing: 

we are deprecating the following:
Command
- `command`
- `command-r-03-2024`
- `command-r-plus-04-2024`

Recommendation for users: Users must move over to newer models, i.e. command-r-08-2024, command-r-plus-08-2024, and/or command-a-03-2025
Endpoints:

- `v1/connectors`
- `v1/chat`
- `v1/generate`
- `v1/summarize`
- `v1/classify`

Recommendation for users: All platform users recommended to move to the v2 API with this change
Other Products & Features: Fine-tuning capabilities across the platform
This means fine-tuning on platform will become unavailable, and meaning that any command-light, command, command-r, rerank, and classify fine-tuned models will become unavailable
command Slack app
Coral Web (Coral.cohere.com)
V1 API Parameters: `connectors` and `search_queries_only` in `/v1/chat`.
